# 约课吧微信小程序

## 项目介绍
约课吧是一款用于生成和管理一周课表的微信小程序，用户可以通过直观的界面选择开课时间并保存。

## 功能特性
- 可视化课表生成界面
- 支持一周七天的课程安排
- 7:00-22:00时间段的灵活选择
- 每30分钟一节课，中间休息5分钟
- 数据同步到后端服务器
- 本地存储备份功能

## 技术架构
- 前端：微信小程序原生开发
- 后端：Spring Boot + Spring Data JPA
- 数据库：H2（开发环境）

## 快速开始

### 1. 后端服务启动
首先需要启动后端服务：
```bash
cd backend
mvn spring-boot:run
```
后端服务将在 `http://localhost:8081` 启动。

### 2. 小程序开发
使用微信开发者工具打开当前项目目录，即可开始开发和预览。

## 使用说明
1. 在课表界面点击对应时间段的格子选择开课时间
2. 再次点击可取消选择
3. 点击右上角"保存课表"按钮保存当前选择
4. 课表将自动同步到后端服务器，并在本地备份

## 注意事项
- 确保后端服务已正确启动
- 小程序需要配置安全域名才能访问后端API
- 开发环境下可以勾选"不校验合法域名"选项

## 开发说明
- 小程序使用原生微信小程序开发框架
- 页面采用响应式设计，适配不同屏幕尺寸
- 数据交互采用RESTful API设计
